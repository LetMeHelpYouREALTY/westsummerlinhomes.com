# Self-hosted fonts

Latin-subset WOFF2 files served from Vercel with long-term cache.

| File | Family | Weights | Source |
|------|--------|---------|--------|
| `inter-latin.woff2` | Inter | 400–700 | [Google Fonts](https://fonts.google.com/specimen/Inter) (OFL) |
| `playfair-latin.woff2` | Playfair Display | 600–700 | [Google Fonts](https://fonts.google.com/specimen/Playfair+Display) (OFL) |

Loaded via `@font-face` in `site.css`. Homepage preloads both files for LCP.
